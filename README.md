# ConsistencyPulse 🔥

A simple, powerful habit tracker to help you stay consistent.  
Built by **Regan Macharia**, Grade 10 student at Nkubu High School, Kenya.

## Features
- Track daily habits with streaks
- Earn points and badges
- Focus timer (Pomodoro)
- Works offline
- Install as an app (PWA)

## Live Demo
[consistency-pulse](https://machariareagan-droid.github.io/Consistency-pulse-/)

## Tech Stack
HTML, CSS, JavaScript (vanilla), Service Worker, LocalStorage
